## 🚀 Как открыть УМК

Учебно-методический комплекс (УМК) размещён в открытом репозитории GitHub.  
Для корректного просмотра рекомендуется использовать приложение **Obsidian**, так как структура материалов ориентирована на работу с ним.

### 📥 Шаг 1. Скачать репозиторий

Перейдите по ссылке:  
👉 [https://github.com/DionisiuBrovka/EC-Instrumental-Software](https://github.com/DionisiuBrovka/EC-Instrumental-Software)

Нажмите кнопку **Code → Download ZIP**, чтобы скачать проект на компьютер.

![Кнопка скачивания репозитория на GitHub](<7 Дополнительные материалы/.readme_src/img_001.png>)

### 💻 Шаг 2. Установить Obsidian

Скачайте и установите Obsidian с официального сайта:  
👉 [https://obsidian.md/](https://obsidian.md/)

![Страница загрузки Obsidian](<7 Дополнительные материалы/.readme_src/img_002.png>)

### 📂 Шаг 3. Открыть хранилище

1. Запустите Obsidian  
2. Выберите **"Открыть папку как хранилище" (Open folder as vault)**  
3. Укажите папку с распакованным репозиторием

![Открытие папки как хранилища в Obsidian](<7 Дополнительные материалы/.readme_src/img_003.png>)

### 🔌 Шаг 4. Включить сторонние плагины

Для корректной работы некоторых элементов:

1. Перейдите в **Settings (Настройки)**  
2. Откройте раздел **Community plugins**  
3. Отключите Safe mode (Безопасный режим)  
4. Включите необходимые плагины

![Включение сторонних плагинов в Obsidian](<7 Дополнительные материалы/.readme_src/img_004.png>)
