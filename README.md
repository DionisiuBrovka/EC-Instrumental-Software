# 📚 УМК «Инструментальное программное обеспечение»

> Учебно-методический комплекс по дисциплине «Инструментальное программное обеспечение».  

---

## 🎯 О дисциплине

Основная цель предмета — изучение принципов и методов разработки, применения и управления программным обеспечением для решения бизнес-задач.

**Задачи курса:**

Изучение синтаксиса Python, типов данных, строк, списков, кортежей, словарей, ножеств, условных операторов, операторов циклов, функций, обработки исключений, модулей, пакетов, аботы с файлами, основ и базовых принципов объектно-ориентированного программирования (далее – ООП), взаимодействия с сетью Интернет, клиент-серверных приложений, фреймворков  библиотек.
