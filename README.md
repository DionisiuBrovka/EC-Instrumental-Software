## 🎯 О дисциплине

Основная цель предмета — изучение принципов и методов разработки, применения и управления программным обеспечением для решения бизнес-задач.

**Задачи курса:**

- Выбор наиболее подходящих технологий и инструментов для конкретных задач
- Разработка и реализация простых и эффективных программных решений
- Управление и оптимизация работы программного обеспечения
- Обзор современных трендов и технологий программирования

## ⚙️ Установка в Obsidian

### Шаг 1 — Скачать хранилище

Скопируйте папку `УМК-ИПО/` в удобное место на компьютере, например:

```bash
~/Documents/Obsidian/УМК-ИПО/
```

### Шаг 2 — Открыть как Vault

1. Откройте **Obsidian**
2. Нажмите **«Open folder as vault»** (Открыть папку как хранилище)
3. Выберите папку `УМК-ИПО/`
4. Obsidian автоматически создаст папку `.obsidian/` с настройками

## 🔗 Навигация и связи

Хранилище использует `[[wiki-ссылки]]` Obsidian для связи между материалами:

- Каждая **лабораторная работа** ссылается на соответствующую **лекцию**
- **ОКР** ссылаются на **варианты**
- Внутри лекций можно ссылаться на связанные темы через `[[]]`

Это создаёт **граф знаний** (Graph View), который показывает взаимосвязи между темами курса.
