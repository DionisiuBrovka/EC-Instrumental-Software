## О дисциплине

- **Основная цель предмета**: изучение принципов и методов разработки, применения и управления программным обеспечением для решения бизнес-задач.
- **Специальность:** 5-04-0612-02 «Разработка и сопровождение программного обеспечения информационных систем»  
- **Уровень образования:** Среднее специальное образование (квалификация специалиста)
